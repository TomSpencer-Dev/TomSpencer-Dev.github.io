Java Servlets

This was a group project that was completed with two other developers enrolled at Algonquin College.

It is a streaming services budget tracking web application. It was built in Eclipse, utilizing Apache Tomcat, Java Servlets, Bootstrap, Animate.css and MySQl.

A user has the ability to register as a new user and login with their password stored in a database in a hashed format. The user manually enters subscription data, and a message appears if they are over their set budget.

React

This was a group project that was completed with two other developers enrolled at Lighthouse Labs.

It is a volunteer sign up and event listing web platform built in React, utilizing Tailwind CSS, Animate.css, PostgreSQL and the Google Calendar API.

A user has the ability to login, logout, sign up for events and delete events. They can also add events to their Google Calendar. You can check it out here → Github

Android

This was a final project I completed while attending Algonquin College.

It is a Guardian News lookup Android phone application. It was built in Android studio and utilizes such features as ListView, Guaradian News API, ToolBar, Drawer etc. SQLite is used for a database, and SharedPreferences is used to store state.

A user has the ability to select a list of news articles from Guardian News based on the category Business, Finance, or Current. They can also do a keywoard search of articles. One can select articles to see more details, or select to add it to their personal saved list. The My List screen displays the articles the user has saved in their list and they can choose to delete them from their list. You can check it out here → Github
